# 个人简历网站

这是一个基于HTML、CSS和JavaScript的个人简历网站，专为电影媒体专业人士设计。

## 功能特点

- 响应式设计，适配各种设备屏幕
- 现代化UI界面，具有动画效果
- 包含个人简介、作品集展示、专业技能和联系方式等板块
- 交互式作品集筛选功能

## 技术栈

- HTML5
- CSS3 (动画、Flexbox、Grid布局)
- JavaScript (原生JS)
- Font Awesome 图标库
- Google Fonts 字体

## 使用方法

1. 克隆仓库到本地
2. 按照`图片替换指南.md`中的说明替换个人照片和作品集图片
3. 修改`index.html`中的个人信息
4. 部署到您喜欢的网络服务器或GitHub Pages
